# Machine Learning: Image Processing & EDA

## Overview
This repository contains programming assignments completed as part of my university coursework in **Machine Learning**. 

The primary focus of these notebooks is handling unstructured image data, performing Exploratory Data Analysis (EDA), and building foundational data processing pipelines from scratch. These projects demonstrate my ability to prepare complex data for machine learning models without relying heavily on high-level, automated ML libraries.

## Key Learning Objectives
* **Data Ingestion & Preprocessing:** Processed raw image datasets using `PIL` (Pillow) and vectorized them into `NumPy` arrays for mathematical manipulation.
* **Exploratory Data Analysis (EDA):** Analyzed and visualized pixel distributions, class imbalances, and dataset characteristics using `Matplotlib`.
* **Feature Engineering:** Implemented scaling, normalization, and image transformations to prepare unstructured data for predictive modeling.
* **Mathematical Foundations:** Applied core linear algebra and statistical concepts to manipulate image data programmatically.

## Technologies Used
* **Language:** Python
* **Core Libraries:** NumPy, Matplotlib, PIL (Pillow)
* **Environment:** Jupyter Notebook

## Repository Contents
* `2024136_A1.ipynb`: Focuses on initial data loading, basic image transformations, and foundational exploratory data analysis.
* `2024136_a2.ipynb`: Expands on the pipeline with advanced vectorization, feature extraction, and preparing the datasets for machine learning algorithms.
